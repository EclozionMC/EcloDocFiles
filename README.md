# EclozionMC Plugin Dependency Documentation

Welcome to the central documentation repository for EclozionMC's internal plugins. This repository contains detailed guides on how to use each plugin as a dependency.

## Core Services
These plugins are foundational and required for most development.

- **[EcloCore](EcloCore_dependency_guide.md)**: The heart of the network. Database connections, command framework, and shared utilities.
- **[EcloSync](EcloSync_dependency_guide.md)**: Universal Data Hub. Cross-server communication (Pub/Sub), Service Discovery, and Data Sync.

## Major Systems
Critical gameplay systems with extensive APIs.

- **[EcloEco](EcloEco_dependency_guide.md)**: Economy system (Coins & BloodMoney).
- **[EcloPerms](EcloPerms_dependency_guide.md)**: Permission management and chat formatting.

## Standard Modules
Standalone features or specific gameplay mechanics.

- **[EcloChat](EcloChat_dependency_guide.md)**: Chat channels and formatting.
- **[EcloCombat](EcloCombat_dependency_guide.md)**: PVP tagging and combat restrictions.
- **[EcloClearLag](EcloClearLag_dependency_guide.md)**: Entity clearing and performance limits.
- **[EcloHDV](EcloHDV_dependency_guide.md)**: Auction House integration.
- **[EcloSpawners](EcloSpawners_dependency_guide.md)**: Spawner mining and stacking.
- **[EcloStaff](EcloStaff_dependency_guide.md)**: Moderation tools (Vanish, Freeze).
- **[EcloTP](EcloTP_dependency_guide.md)**: Teleportation (Warps, Spawns).
- **[EcloVoteParty](EcloVoteParty_dependency_guide.md)**: Voting rewards and events.
