# EcloVoteParty Dependency Guide

> **Version**: 1.0.0
> **Type**: Voting Events
> **Importance**: STANDARD

## 1. Introduction
**EcloVoteParty** manages rewards when a global vote count is reached.

## 2. Integration
Currently standalone.

### Maven
```xml
<dependency>
    <groupId>com.eclozion.voteparty</groupId>
    <artifactId>EcloVoteParty</artifactId>
    <version>1.20</version>
    <scope>provided</scope>
</dependency>
```
